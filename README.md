# 📊 ShopEasy Marketing Analytics Dashboard

A comprehensive Marketing Analytics project built using **SQL Server** and **Power BI** to analyze ShopEasy's marketing performance, customer behavior, engagement, and product performance.

---

## 📌 Project Overview

This project aims to help the marketing team improve campaign performance by analyzing customer journeys, engagement metrics, conversion rates, customer feedback, and product performance.

The dashboard provides actionable insights to support data-driven marketing decisions and optimize business performance.

---

## 🎯 Business Objectives

- Increase Conversion Rate
- Improve Customer Engagement
- Analyze Customer Feedback
- Identify Top Performing Products
- Support Marketing Decision Making

---

## 🛠️ Tools & Technologies

- SQL Server
- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Modeling

---

## 📂 Dataset

The project is based on the **ShopEasy Marketing Analytics** dataset.

### Tables

- Customers
- Products
- Customer Journey
- Customer Reviews
- Engagement Data

---

## 🧹 Data Cleaning

The dataset was cleaned using SQL Server before importing into Power BI.

Cleaning tasks included:

- Removing duplicate records
- Handling missing values
- Creating SQL Views
- Standardizing data
- Splitting combined columns
- Preparing data for reporting

3. Open the `.pbix` file in Power BI Desktop.
4. Refresh the data model.
5. Explore the interactive dashboard.
